# open-edx-engaged-theme
EngagED custom theme for Open edX

## Installation

```bash
git clone https://github.com/engaged-edu/open-edx-engaged-theme.git "$(tutor config printroot)/env/build/openedx/themes/"
tutor images build openedx
```
